	- Access Control List
		- Requires `acl` option during mount
		- `setfacl TYPE:[NAME]:PERM`: Sets ACL on files and directories
			- `-m`: Modifies instead of set
		- `getfacl`: Gets file acl statis
	- Directory owner has ability to delete files even if file is owned by others

			- `IPADDR`: Default gateway on none
