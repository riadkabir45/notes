- Unix users, password and group files #Light
	- `/etc/passwd`
		- Rows `username,x,userid,groupid,name,homedir,shell`
		- The x tells the password is stored in shadow file
		- `/sbin/nologin` denies login, mainly for service related users
	- `/etc/shadow`
		- Stores the password hash of each user and contains !! for blank passwords
	- `/etc/group`
		- Rows `groupid,x,users`
	- `useradd USERNAME`
		- `-s`: Default shell
		- `-d`: Home dir
		- `-m`: Populate home dir
